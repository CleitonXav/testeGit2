Crie o arquivo .txt:

Para criar o arquivo de texto, você pode usar o comando echo ou o type nul >.

°Para criar um arquivo com conteúdo específico, digite:
no cmd
echo Este é o conteúdo do arquivo > arquivo.txt

Isso criará um arquivo chamado arquivo.txt com o texto "Este é o conteúdo do arquivo".


°Para criar um arquivo vazio, digite:
no cmd
type nul > arquivo.txt

Esse comando cria um arquivo vazio chamado arquivo.txt.
